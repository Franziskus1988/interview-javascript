Task: Calculate string in the input box, check if the squared braces are balanced.

Request:
1. Create a function calls "isBalanced(string)" to check the value typed in the input box
2. Display anywhere a text "true" with green color if the string is balanced, otherwise display "false" with red color.

Balanced examples: 
 hello [world]
 [[world[]]]
 a[]b[]c[]
 aaaa

Non-balanced: 
 ]hello[
 world [[]
 []]
